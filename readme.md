# 練習D3
## Taiwan-Town.html
從[政府資料開放平台](https://data.gov.tw/)取得鄉鎮市區界線檔案 mapdata202203151020.zip，拖曳至[mapshape 網站](https://mapshaper.org/)，轉換得 Topojson 檔案 TOWN_MOI_1100415.json，載入Taiwan-Town.html，顯示北北基的行政地區圖。
